Legacy Project Chicago

Legacy Project Chicago is a maintenance project that required updating from Drupal 9 to 10.

During the transition from Drupal 9 to 10, there were permission issues. There were many modules that fueled this issue, but we resolved it by making a custom module that overrode the other modules issues. 

This upgrade took close to 60 hours and was extensively tested for quality assurance.

Sjogren’s Foundation

Sjogren’s Foundation was in need of a Drupal update and some key features for the site.

During the transition from Drupal 8 to Drupal 10, there were many difficulties that had to do a lot with undoing confusing code and not good practice code from previous developers. To fix these issues, we did a lot of testing and made careful and thoughtful changes.

The site is stable now and we’ve moved from maintenance to working on new features for the site. Our work to refactor and update build customer trust and offered the opportunity to start work on many new features.
