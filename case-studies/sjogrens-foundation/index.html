Update and Maintenance

Sjogren’s Foundation was in need of a Drupal update and some key features for the site. During the transition from Drupal 8 to Drupal 10, there were many difficulties that had to do a lot with undoing confusing code and not good practice code from previous developers. To fix these issues, we did a lot of testing and made careful and thoughtful changes. 

 

The site is stable now and we’ve moved from maintenance to working on new features for the site. Our work to refactor and update build customer trust and offered the opportunity to start work on many new features.

  • Maintenance work
  • Website redesign
  • New integrations
  • New custom features
